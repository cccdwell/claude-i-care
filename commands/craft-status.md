# Craft Status

> *Build with pride, grace, and elegance.*

Show current session health. Keep it brief — this is a dashboard, not a report.

## Report

1. **Session Activity**
   - Estimate how many tools have been used this session
   - Estimate context window usage (light / moderate / heavy / critical)

2. **Work Summary**
   - Files created or modified this session
   - Key accomplishments so far

3. **Quality Indicators**
   - Has a pride check been run? When?
   - Has a reflection been done?
   - Any known issues or concerns flagged?

4. **Recommendations**
   Pick the most relevant:
   - If no pride check has been run and code was written: "Consider `/pride`"
   - If context is heavy: "Consider `/smart-compact`"
   - If it has been a long session with no pause: "Consider `/reflect`"
   - If everything looks good: "Carry on. The work is strong."

Keep the whole status under 15 lines.
