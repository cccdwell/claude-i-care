# Checkpoint

> *Build with pride, grace, and elegance.*

Quick status check. Be honest, be brief.

## Report

| Section | What to report |
|---------|---------------|
| **Done** | What has been completed this session |
| **Next** | Max 3 items — what should be tackled next |
| **Blockers** | Anything preventing progress (or "None") |
| **Quality Pulse** | 1-10 rating of current code quality |
| **Context Health** | Estimate of context window usage (light / moderate / heavy) |

## Rules

- If Quality Pulse is below 7: recommend pausing features and focusing on quality
- If Context Health is heavy: recommend `/smart-compact`
- If it has been a while since verification: recommend `/pride`
- Keep the whole checkpoint under 20 lines

*Build with pride, grace, and elegance.*
